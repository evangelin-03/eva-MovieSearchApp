# Movie Search App

A responsive movie search application that allows users to search for movies and view results using the OMDb API.

## Live Demo
eva-movie-search-app-5xho.vercel.app

## Features
- Search movies by title
- Display movie posters and release year
- Pagination to browse multiple results
- Responsive layout for different screen sizes
- Dynamic API data fetching

## Technologies Used
- HTML
- CSS
- JavaScript
- OMDb API

## How It Works
The application sends a request to the OMDb API when a user searches for a movie title.  
The API returns movie data which is dynamically displayed on the page.
